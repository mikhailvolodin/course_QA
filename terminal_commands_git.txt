1) Скачать веб-репозиторий на ПК - git clone + SSH или HTTPS код
2) Проверить сходство локального репозитория с веб - git status
3) Сделать отслеживание файла - git add + имя файла
4) Сделать отслеживание всех файлов - git add .
5) Фотография состояния гита на ПК - git commit
6) Фотография состояния гита на ПК + комментарий изменений - git commit -m
7) Добавить отслеживание, фото состояния и написать комментарий - git commit -am
8) Отправить файлы на внешний репозиторий - git push
9) Получить информацию о состоянии внешнего репозитория - git fetch
10) Скачать состояние внешнего репозитория на локальный - git pull
