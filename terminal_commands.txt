1) Посмотреть где я — pwd 
2) Создать папку — mkdir имя_папки
3) Зайти в папку — cd название_папки
4) Создать 3 папки — mkdir folder_1 folder_2 folder_3
5) Зайти в любою папку - cd путь к нужной папке
6) Создать 5 файлов (3 txt, 2 json) - touch file_1.txt file_2.txt 
   file_3.txt file_4.json file_5.json
7) Создать 3 папки - mkdir d_1 d_2 d_3
8) Вывести список содержимого папки — ls (возможны варианты ls -a; ls -l; 
   ls -la)
9) + Открыть любой txt файл — если имеется ввиду текстовый редактор, то мы 
   используем vim 
10) + написать туда что-нибудь, любой текст — нажимаем i и пишем ручками текст
11) + сохранить и выйти — нажимаем Esc : wq enter
12) Выйти из папки на уровень выше - cd ..
13) Переместить любые 2 файла, которые вы создали, в любую другую 
    папку — mv имя_файла имя_файла имя_папки_куда_перемещать/  
14) Скопировать любые 2 файла, которые вы создали, в любую другую 
    папку — cp имя_файла имя_файла имя_папки_куда_перемещать/
15) Найти файл по имени - find -name имя_файла (можно использовать флаг -i, что бы убрать чувствительность к регистру)
16) Просмотреть содержимое в реальном времени — tail -f имя_файла 
17) Вывести несколько первых строк из текстового файла — head -1 
    имя_файла : выведет первую строку; head -2 имя_файла : выведет 
    первые две строки; и т. д.
18) Вывести несколько последних строк из текстового файла — tail -1 
    имя_файла : выведет последнюю строку; tail -2 имя_файла : выведет две 
    последние строки; и т. д.
19) Просмотреть содержимое длинного файла - less (команда grep — поможет найти текст или символы в указанном файле или 
    папке)
20) Вывести дату и время — date (возможны варианты date +"%D %T", timedatectl )
